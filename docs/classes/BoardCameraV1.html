Class BoardCameraV1

This is the backbone of the canvas panning and a lot of other stuff that is made easy by the camera. This would normally be retrieved from the board class as this camera is not that useful without the board.

Example Usage: 

const camera = board.getCamera();
camera.setPosition({x: 0, y: 0});

  • convert2WorldSpaceWRT(centerPoint, interestPoint): Point

  • Convert a point that is relative to the center point in the view port space to the world space. For example if you have a point and you want to know that the world space coordinate of a point that appears to the right of the point in the view port space. (Interest point is relative to the center point in the view port space) The center point is in world space.
